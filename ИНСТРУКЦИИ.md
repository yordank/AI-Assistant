# 🤖 Telegram AI Bot - Инструкции за инсталация (Node.js)

## 📋 Какво прави ботът?

✅ **Чат с AI** - Просто пишеш текст и той отговаря (като ChatGPT)
✅ **Запомня историята** - Помни предишните ви разговори
✅ **Периодични задачи** - Автоматично изпълнява задачи на интервали (проверки на jobs.bg, новини и др.)
✅ **Без команди** - Не е нужно да пишеш /start, /help и т.н. - просто текст!

---

## 🚀 Стъпка 1: Инсталация на Node.js

### Windows:
1. Свали Node.js от https://nodejs.org
2. Избери LTS версията (препоръчва се)
3. Инсталирай (оставяй default настройките)

### Mac:
```bash
# С Homebrew
brew install node

# Или свали от nodejs.org
```

### Linux (Ubuntu/Debian):
```bash
# Ubuntu/Debian
curl -fsSL https://deb.nodesource.com/setup_lts.x | sudo -E bash -
sudo apt-get install -y nodejs

# Или с nvm (препоръчително)
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash
nvm install --lts
```

**Провери инсталацията:**
```bash
node --version    # Трябва да покаже v18.x или по-нова
npm --version
```

## 📦 Стъпка 2: Инсталирай библиотеките

Отвори терминал/CMD в папката с бота и изпълни:

```bash
npm install
```

Това ще инсталира:
- `telegraf` - Telegram Bot API
- `@anthropic-ai/sdk` - Claude AI
- `dotenv` - Управление на environment variables

## 🔑 Стъпка 3: Конфигурация

### Вариант А: С .env файл (ПРЕПОРЪЧИТЕЛНО)

1. Копирай примерния .env файл:
```bash
# Windows CMD
copy .env.example .env

# Mac/Linux/Windows PowerShell
cp .env.example .env
```

2. Редактирай `.env` файла:
```
TELEGRAM_TOKEN=123456:ABC-DEF1234ghIkl-zyx57W2v1u123ew11
ANTHROPIC_API_KEY=sk-ant-api03-xxxxxxxxxxxx
```

### Вариант Б: Директно в кода

Отвори `bot.js` и намери ред 8-9:

```javascript
const TELEGRAM_TOKEN = 'твоят_telegram_token_тук';
const ANTHROPIC_API_KEY = 'твоят_anthropic_ключ_тук';
```

## ⏰ Стъпка 4: Настрой периодични задачи (опционално)

В `bot.js` намери `PERIODIC_TASKS` (около ред 11):

```javascript
const PERIODIC_TASKS = [
    {
        name: 'Проверка заплати',
        intervalMinutes: 60,  // На всеки час
        prompt: 'Провери jobs.bg за средната заплата в Пловдив и ако има промяна над 50 лв спрямо последната проверка, кажи ми.',
        enabled: false  // Смени на true за да активираш
    },
];
```

**За да активираш задача:**
- Смени `enabled: false` на `enabled: true`

**За да добавиш нова задача:**
```javascript
{
    name: 'Име на задачата',
    intervalMinutes: 30,  // Колко често (в минути)
    prompt: 'Какво да прави AI-то',
    enabled: true
},
```

**Примери:**
```javascript
// Проверка на криптовалути на всеки 15 мин
{
    name: 'Крипто цени',
    intervalMinutes: 15,
    prompt: 'Провери текущата цена на Bitcoin и Ethereum и ми кажи само ако има промяна над 2%',
    enabled: true
},

// Дневно напомняне
{
    name: 'Здравно напомняне',
    intervalMinutes: 60,
    prompt: 'Напомни ми да пия вода и да направя почивка',
    enabled: true
},
```

## ▶️ Стъпка 5: Стартирай бота

### Нормален старт:
```bash
npm start
```

### Development режим (с auto-restart):
```bash
npm run dev
```

### Или директно:
```bash
node bot.js
```

### Трябва да видиш:
```
Starting Telegram AI Bot...
Task scheduler started
No active periodic tasks (edit PERIODIC_TASKS to enable)
Bot is running! Send a message in Telegram to start.
```

## 💬 Стъпка 6: Използвай бота

1. Отвори Telegram
2. Намери твоя бот (username-а който си му дал в @BotFather)
3. **Просто му пиши** - не е нужно /start
4. При първо съобщение ще се регистрираш като admin

### Примери какво да пишеш:

```
Какво е средната заплата в Пловдив?

Направи ми план за тренировка

Обясни ми как работи квантовият компютър

Какви са новините днес?
```

### Специални текстове (не са команди):

```
изчисти история        → Започва нов разговор
статус бот            → Показва статус и активни задачи
нов разговор          → Изчиства историята
```

## 🌐 Стъпка 7: Пусни в облак (опционално)

За да работи 24/7 без да държиш компютъра включен:

### Railway.app (Препоръчително):

1. Създай акаунт на https://railway.app
2. Свържи GitHub акаунт
3. "New Project" → "Deploy from GitHub repo"
4. Изкачи кода в GitHub repo
5. В Railway добави Environment Variables:
   - `TELEGRAM_TOKEN` = твоят token
   - `ANTHROPIC_API_KEY` = твоят API ключ
6. Railway автоматично ще detectне Node.js и ще стартира с `npm start`

### Render.com:

1. Създай акаунт на https://render.com
2. "New" → "Web Service"
3. Свържи GitHub repo
4. Build Command: `npm install`
5. Start Command: `npm start`
6. Добави Environment Variables

### Fly.io:

```bash
# Инсталирай flyctl
curl -L https://fly.io/install.sh | sh

# Login
flyctl auth login

# Deploy
flyctl launch
flyctl deploy
```

### Heroku:

1. Създай `Procfile`:
```
worker: node bot.js
```

2. Deploy:
```bash
heroku create
git push heroku main
heroku ps:scale worker=1
```

## ❓ Чести проблеми

### "Cannot find module 'telegraf'"
➡️ Изпълни: `npm install`

### "Unauthorized" или "401" от Telegram
➡️ Провери дали Telegram token-а е правилен в .env

### "Authentication failed" от Anthropic
➡️ Провери дали Anthropic API ключа е правилен

### Ботът не отговаря
➡️ Провери дали скриптът работи (виж терминала)
➡️ Провери internet connection
➡️ Провери дали `.env` файлът е в същата папка като `bot.js`

### "Error: .env file not found"
➡️ Създай `.env` файл или редактирай директно в `bot.js`

### Автоматичните задачи не работят
➡️ Провери дали `enabled: true` в PERIODIC_TASKS
➡️ Виж терминала за грешки

## 💡 Съвети

1. **Тествай първо локално** преди да пуснеш в облак
2. **Използвай .env файл** - не commit-вай ключовете в Git
3. **Добави .gitignore:**
```
node_modules/
.env
```
4. **Следи разходите** - Anthropic API струва пари след $5 безплатен кредит
5. **Използвай nodemon за development** - `npm run dev`
