# 🤖 Telegram AI Bot (Node.js)

Прост Telegram бот с Claude AI - **без команди**, просто пиши текст!

## ✨ Функции

- 💬 **Чат с AI** - Просто пиши и той отговаря
- 🧠 **Запомня историята** - Помни целия разговор
- ⏰ **Периодични задачи** - Автоматични проверки и напомняния
- 🚫 **Без команди** - Не е нужно `/start`, `/help` и т.н.

## 🚀 Бърз старт

```bash
# 1. Инсталирай dependencies
npm install

# 2. Конфигурирай ключовете
cp .env.example .env
# После редактирай .env файла

# 3. Стартирай
npm start
```

## 📖 Пълни инструкции
