# 🚀 БЪРЗ СТАРТ (Node.js)

## 1️⃣ Инсталирай Node.js:

Ако нямаш Node.js: https://nodejs.org (Свали LTS версията)

## 2️⃣ Инсталирай библиотеките:

```bash
npm install
```

## 3️⃣ Конфигурирай ключовете:

**Вариант А:** Създай `.env` файл (препоръчително):

```bash
cp .env.example .env
```

После редактирай `.env` файла:
```
TELEGRAM_TOKEN=твоят_telegram_token_тук
ANTHROPIC_API_KEY=твоят_anthropic_ключ_тук
```

**Вариант Б:** Редактирай директно `bot.js` (ред 8-9):

```javascript
const TELEGRAM_TOKEN = 'твоят_telegram_token_тук';
const ANTHROPIC_API_KEY = 'твоят_anthropic_ключ_тук';
```

## 4️⃣ Стартирай:

```bash
npm start
```

или за development с auto-restart:

```bash
npm run dev
```

## 5️⃣ Използвай:

Отвори Telegram → намери твоя бот → просто пиши!

---

## ⏰ За периодични задачи:

В `bot.js` намери `PERIODIC_TASKS` (ред 11) и смени:

```javascript
enabled: false   →   enabled: true
```

---

## 📝 Примерни задачи:

```javascript
// Проверка заплати на всеки час
{
    name: 'Jobs.bg',
    intervalMinutes: 60,
    prompt: 'Провери jobs.bg за нови обяви в Пловдив',
    enabled: true
},

// Дневни новини
{
    name: 'Новини',
    intervalMinutes: 1440,  // 24 часа
    prompt: 'Кратко резюме на новините в България',
    enabled: true
},

// Цени крипто
{
    name: 'Криpto',
    intervalMinutes: 30,
    prompt: 'Цена на Bitcoin и Ethereum',
    enabled: true
},
```
